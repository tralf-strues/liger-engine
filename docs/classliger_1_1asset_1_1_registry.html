Detailed Description

Registry of assets contained in an asset folder.

Manages mapping from asset uuids to their physical file paths. All this information is also gets saved to the corresponding registry file.

Example structure of an asset folder:

assets/
.lregistry
textures/
player/
player_albedo.png
player_normal.png
materials/
player.lmat
meshes/
player.lmesh
scenes/
scene0.lscene
sounds/
player_hello.mp3
player_goodbye.mp3

Example contents of the corresponding .liger-registry file:

- file: textures/player/player_albedo.png
id: 0x7449545984958451
- file: textures/player/player_normal.png
id: 0x2435204985724523
- file: materials/player.lmat
id: 0x9208347234895237
