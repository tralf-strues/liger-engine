virtual uint32_t liger::rhi::ITexture::CreateView ( const TextureViewInfo &  info)
pure virtual

Create a view for the texture.

Note
The indexing of views is monotonous. Upon texture creation, a default view is created with index kTextureDefaultViewIdx = 0, and after that each consecutive call to this method returns the next integer.
