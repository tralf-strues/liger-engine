Detailed Description

Importer interface.

Importers are basically converters of one asset file (based on extension) to possibly several asset files better suited for some particular usage. An example can be an FBX importer that generates mesh/material/texture/etc. files in a format which is tailored specifically for a particular renderer implementation.
